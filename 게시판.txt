JSP 게시판 만들기
	기능
1. 로그인 기능 (세션 사용)
 1.1 로그 아웃 
 1.2 로그인 후 아무런 동작이 없을 경우 자동 로그아웃 (세션 유지 시간)

2. 회원가입 (회원 테이블에 데이터 추가)
 2.1 회원 정보 보기
 2.2. 회원 정보 수정
 2.3 회원 탈퇴
_---------
3. 글 전체 내용 보기
 3.1 테이블로 게시글 출력, 글 번호, 글 제목, 글쓴이, 조회수, 등록시간 출력 (오름차순으로 정렬)
 3.2 로그인과 상관없이 글 목록은 볼 수 있음.

4. 글 상세 보기
 4.1 페이지 이동 후 글 상세내용 출력
 4.2 글 번호, 글 제목, 글 내용, 글쓴이, 등록시간, 조회 수 출력
 4.3 로그인과 상관없이 글 내용 확인 가능 

5. 글 쓰기 
 5.1 로그인 시에만 글쓰기 가능
 5.2 글 제목, 글 내용 입력 후 글 등록 가능
------------
6. 글 수정
 6.1 로그인 시에만 글 수정 가능
 6.2 해당 글쓴이 만 수정 가능

7. 글 삭제
 7.1 로그인 시에만 글 삭제 가능
 7.2 해당 글쓴이 만 삭제 가능
 7.3 관리자는 글 삭제 가능

게시판
글 번호 (정수, 자동증가, 기본키 )  	- 컬럼명: board_idx
제목 (문자열 100자까지, not null) 	- 컬럼명: board_title
내용 (문자열 500자, not null) 	- 컬럼명: board_contents
글쓴이 (문자열 45자, not null) (사용자 ID) - 컬럼명: board_member_id
글 비밀번 (문자열, 45자. , not null)	- 컬럼명: board_pw
조회수 (정수, 기본값 0 )		- 컬럼명: board_cnt
등록날짜 (datetime, not null)		- 컬럼명: board_reg_date
수정날짜 (datetime, null)		- 컬럼명: board_up_date

사용자
사용자 번호 : 정수,기본키,자동 증가)	- 컬럼명: member_idx
사용자 ID : 문자열 45, not null, 유니크)	 - 컬럼명: member_id
사용자 PW : 문자열 45자, not null)	- 컬럼명: member_pw
사용자 이름 : 문자열 45자,  not null	- 컬럼명: member_name
사용자 이메일 : 문자열 100자, not null	- 컬럼명:  member_email
사용자 전화번호 : 문자열 11자, null  - 컬럼명: member_phone
사용자 주소 : 문자열 200자, null  - 컬럼명: member_addr	
사용자 등급 : 정수 1자, 기본값 0/  # 0,1  : 일반 사용자, 1: 관리자
